1. Lancez `npm run dev`
2. Explorez la landing page
3. Accédez au dashboard (`/app`)

## ❓ Problèmes courants

### Le port 3000 est déjà utilisé
Modifiez le port dans `vite.config.js` (ligne 5)

### Erreur "Cannot find module"
```bash
rm -rf node_modules package-lock.json
npm install
```

### L'application ne se lance pas
1. Vérifiez que Node.js est installé : `node --version`
2. Version minimale requise : Node 18+
3. Réinstallez les dépendances

## 💡 Conseils

- **Sauvegarde automatique** : Vite recharge automatiquement à chaque modification
- **Console développeur** : Ouvrez F12 pour voir les erreurs éventuelles
- **Hot reload** : Pas besoin de recharger la page manuellement
